# ROBO RUMBLE ULTIMATE Web Release

Publisher: RR Apps and Games

## Build

```bash
npm run build
```

The website files are generated in `dist/`.

## Local Production Preview

```bash
npm run preview
```

## Hosting

Upload the full contents of `dist/` to a static web host such as Netlify, Vercel, Cloudflare Pages, GitHub Pages, or a standard web hosting account.

The Vite build uses relative asset paths, so the game can run at a root domain or a subfolder, for example:

```text
https://rrappsandgames.com/
https://rrappsandgames.com/robo-rumble/
```
